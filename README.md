## meta-test-be  

추가 테스트!!!
### 프로젝트 설정
```js
git clone https://github.com/kode-krew/meta-test-be.git

```

### .env 설정
```js
touch ./.env
# 'env-example' 파일을 참고하여 '.env' 설정
```

### 프로젝트 설치
```js
npm install

```

### 시작하기
```js
npm run start:dev

```

### Swagger UI
```js
http://127.0.0.1:8000/api

```
